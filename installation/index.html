安装指南¶

manim安装并不难，只需要解决相关依赖就好了：

  1. python及模块

  2. ffmpeg

  3. LaTeX发行版

  4. sox（可选）

一些推荐的中文安装类教程：

  • 一视数学卷毛杨的专栏教程

  • 一视数学卷毛杨的互动版视频教程

  • pdcxs的专栏教程

  • pdcxs关于vscode自动运行脚本的教程

关于安装后在IDE中颜色会提示错误，尝试按照 constants页面最下方 处理
